# 🏠 SCT_ML_1 - House Price Prediction using Linear Regression

## 📌 SkillCraft Technology – Machine Learning Internship

### 📖 Project Overview
This project is developed as part of the SkillCraft Technology Machine Learning Internship. The goal is to build a Linear Regression model that predicts house prices based on various housing features. The project covers the complete machine learning pipeline, including data preprocessing, exploratory data analysis, model building, evaluation, and prediction.

---

## 🎯 Objective
The primary objective of this project is to develop a machine learning model that accurately predicts house prices using Linear Regression. The model learns the relationship between different housing features and the target variable (price), enabling reliable price estimation.

---

## 📂 Dataset
The project uses the Housing Price Prediction dataset, which contains multiple attributes describing residential properties. The dataset includes both numerical and categorical variables that influence house prices.

### Features
- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status
- Price (Target Variable)



## 🛠️ Technologies and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook


## 📊 Project Workflow

### 1. Data Collection
- Load the housing dataset.

### 2. Data Preprocessing
- Check missing values.
- Handle categorical variables.
- Encode categorical features.
- Prepare data for training.

### 3. Exploratory Data Analysis (EDA)
- Analyze feature distributions.
- Identify relationships between variables.
- Visualize important trends using plots.

### 4. Feature Selection
- Select relevant input features for prediction.

### 5. Train-Test Split
- Split the dataset into training and testing sets.

### 6. Model Building
- Train a Linear Regression model using Scikit-learn.

### 7. Model Evaluation
Evaluate the model using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

### 8. Prediction
- Predict house prices for unseen data.
- Compare predicted values with actual values.



## 📈 Results
The Linear Regression model successfully predicts house prices using the available housing features. The model demonstrates a strong relationship between property characteristics and house prices and provides accurate predictions on the test dataset.
